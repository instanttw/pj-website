== Installation ==

= Automatic Installation =

1. Log in to your WordPress dashboard
2. Navigate to Plugins > Add New
3. Search for "Essential Addons for WPBakery Page Builder"
4. Click "Install Now" and then "Activate"
5. Go to EA WPBakery dashboard to configure

= Manual Installation =

1. Download the plugin zip file
2. Log in to your WordPress dashboard
3. Navigate to Plugins > Add New > Upload Plugin
4. Choose the downloaded zip file
5. Click "Install Now" and then "Activate"
6. Go to EA WPBakery dashboard to configure

= After Installation =

1. Ensure WPBakery Page Builder is installed and activated
2. Navigate to **EA WPBakery** in your WordPress admin menu
3. Configure your preferred settings
4. Enable/disable elements as needed
5. Start building with 538+ elements!

== Frequently Asked Questions ==

= Does this plugin require WPBakery Page Builder? =

Yes, Essential Addons for WPBakery Page Builder requires WPBakery Page Builder to be installed and activated. The plugin will display an admin notice if WPBakery is not detected.

= Can I disable elements I don't use? =

Absolutely! Navigate to EA WPBakery > Elements and toggle any element on/off. This improves performance by not loading unused element assets.

= Is the plugin translation ready? =

Yes, the plugin is fully translation ready with all strings properly internationalized. You can translate it using .po files or services like Loco Translate.

= Does it work with multisite? =

Yes, the plugin is fully compatible with WordPress multisite installations, supporting both network-wide and per-site activation.

= How do I improve performance? =

1. Disable unused elements in EA WPBakery > Elements
2. Enable conditional asset loading in Settings
3. Use the custom analytics table (Session 6)
4. Run database optimization in Performance page
5. Enable caching on your server

= Is it compatible with my theme? =

Yes, Essential Addons for WPBakery Page Builder works with any WordPress theme that supports WPBakery Page Builder.

= Can I use it on commercial projects? =

Yes, the plugin is released under GPLv3 license, allowing use on unlimited personal and commercial projects.

== Privacy Policy ==

Essential Addons for WPBakery Page Builder does not collect or store any personal data. The plugin:

* Does NOT connect to external services
* Does NOT track user behavior
* Does NOT send data to third parties
* Stores only configuration data locally

Analytics tracking (if enabled) only tracks element usage on your own website and stores data in your WordPress database.
